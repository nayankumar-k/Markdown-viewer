# Markdown Preview

Chrome extension that renders markdown files directly in your browser.

## How to Use

1. Go to `chrome://extensions/`
2. Enable **Developer mode** (top-right)
3. Click **Load unpacked**
4. Select this folder
5. Open any `.md` file in the browser

## Supported URLs
- `file:///path/to/file.md` (local files)
- `http(s)://domain/file.md` (web URLs)

## Features
- 📄 View .md files directly in browser
- 🎨 Syntax highlighting
- 😊 Emoji support
- 📱 Responsive design
